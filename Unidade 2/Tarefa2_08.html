Exemplo de lista de Descrição

TCP
O Protocolo de Controle de Transmição (TCP) é um metodo (protocolo) usado junto com protocolos da internet (ip) para enviar dados na forma de unidades de mensagens, chamadas de pacotes, entre computadores pela internet.
IP
O Protocolo da Internet (IP) é o método ou protocolo pelo qual os dados são enviados de um computador para o outro na internet. Cada computador na internet é indentificado de forma unica por um endereço ip
FTP
O Protocolo de tranferencia de arquivos (FTP) é um protocolo usado para trocar arquivos entre computadores na internet
HTTP
O Protocolo de tranferencia de hipertexto (HTTP) é o protocolo usado para trocar texto, imagens graficas, som, video e outros arquivos multimidia na web
